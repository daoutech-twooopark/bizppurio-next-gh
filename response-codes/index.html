응답 코드

서비스별 결과 코드입니다. 응답 본문의 code 필드가 처리 결과를 나타내며(1000=성공), 발송 후 통신사·카카오·RCS·네이버가 반환하는 채널별 전송 결과 코드는 각 서비스 탭의 품목별 표를 참고하세요.

HTTP

비즈뿌리오 API는 인증 오류(4xx)와 Rate Limit(429)를 제외하면 HTTP 상태를 따로 명시하지 않고 200으로 응답합니다. 실제 성공/실패는 응답 본문의 code 값으로 판단하세요.

공통SMSLMS / MMS카카오 비즈메시지RCS네이버 톡톡

메시지 API 응답에서 사용하는 결과 코드입니다. 공통(API 호출 자체의 응답 상태·결과 코드)과, 통신사·카카오·RCS·네이버가 응답한 품목별 전송 결과 코드(API 응답의 report[].result 또는 Webhook의 RESULT에 기록)로 나뉩니다.

공통

BIZAPI 응답 본문의 code 필드는 비즈뿌리오 자체 결과 코드이며, API 호출 자체의 처리 결과(인증·요청 검증 등)를 나타냅니다. HTTP 상태 코드와 다음과 같이 매핑됩니다.

NOTE: 메시지 발송 후 통신사·카카오·RCS·네이버에서 받은 전송 결과 코드(4100 / 6600 / 7000 / 8000 / 5000 등)는 아래 품목별 표를 참고하세요.
